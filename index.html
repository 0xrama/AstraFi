About AstraFi

AstraFi is a next-generation decentralized finance (DeFi) platform built on Soundness. It allows users to deposit and stake $SND tokens securely while earning rewards. The platform is designed to be simple, transparent, and accessible for both new and experienced users.

Features

  • Secure token staking with real-time reward calculation.
  • User-friendly dashboard to track deposits and rewards.
  • Transparent management of funds through Soundness smart contracts.
  • Responsive, professional UI with interactive experience.

Motivation

DeFi can be complex and intimidating. AstraFi simplifies the process by offering a secure, intuitive, and accessible staking platform, leveraging Soundness to manage blockchain operations efficiently.

Tech Stack

  • Blockchain: Soundness CLI
  • Smart Contracts: ERC-20 for staking and rewards
  • Backend: Node.js
  • Frontend: HTML, CSS, JavaScript

Roadmap

  1. Alpha Launch: Implement basic staking and deposit functionality.
  2. Dashboard Enhancements: Add charts, visual feedback, and analytics.
  3. Automated Rewards: Implement reward distribution using Soundness features.
  4. Community Feedback: Iterate and improve features based on user input.
